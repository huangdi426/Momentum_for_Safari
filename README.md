# 插件配置

## 1. 打开 Safari 上的“开发”菜单 
打开**偏好设置**，点击**高级**
![最下面的一栏打上勾](https://github.com/huangdi426/Momentum_for_Safari/blob/master/img/1.png)

## 2. 添加扩展
点击**开发**，选择**显示扩展构建器**。
![kuozhan](https://github.com/huangdi426/Momentum_for_Safari/blob/master/img/2.png)
击左下角的 **+** 号，点击**添加扩展**，选择该文件夹。
